# 路线图与方向

以下内容描述**方向性**规划，具体排期与实现以仓库内 `CHANGELOG` 与代码为准。

## 当前重心

- 文件类备份/恢复的稳定性与可观测性
- gRPC 链路上的 TLS / 网关集成能力
- S3 数据面：大对象分片、恢复流式与可配置阈值

## 可能扩展

- 更多数据源插件（数据库类等）需在架构上保持与控制面/Agent 边界一致
- 企业场景下的审计、限流与身份体系深化

若你参与贡献，欢迎从 [本地开发环境](../development/local-setup.md) 起步，并关注发版说明。

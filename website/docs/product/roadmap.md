# 路线图

产品能力范围与交付节奏以当前分支 **`CHANGELOG.md`** 与代码为准。长期目标：**可商业化 SaaS**，持续加强多租户运营、安全合规、可观测性与集成体验。

## 当前重心

- 文件类备份/恢复的 **稳定性、完整性与可观测性**（指标、告警、恢复演练）。
- **gRPC 入口** 与 TLS / 网关 / mTLS 等企业集成路径。
- **S3 数据面**：大对象分片、续传、加密与 **按租户键前缀** 隔离；连接解析自 **`storage_profiles`**（AssumeRole / 静态密钥）。

## 演进方向

- **订阅与商业化**：配额、套餐、计费信号与运营工具链（与现有 Prometheus 用量类指标衔接）。
- **身份与企业集成**：OIDC 场景深化、细粒度策略与审计报表。
- **更多数据源插件**：在保持「Agent 不连元数据库、大对象只走对象存储」前提下扩展插件类型。
